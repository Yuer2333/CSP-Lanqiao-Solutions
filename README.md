## 🛠️ 编码规范

本仓库所有源码均遵循以下工程规范：
1. **防爆流：** 涉及计数与大数累加时，一律使用 `long long` 防止 int 溢出。
2. **拒绝硬编码：** 数组大小使用 `const int N = 1e5 + 7;` 统一管理。
3. **见名知意：** 变量与函数命名告别 `a`, `b`, `c`，尽量采用具有物理意义的英文单词。
